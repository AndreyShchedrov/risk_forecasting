Андрей Щедров

## Руководитель

## Тема работы
Прогнозирование риска в страховании жизни с применением методов машинного обучения

## Саммари
Исследование проводилось на данных Российского мониторинга экономического положения и здоровья населения за 2013–2023 годы
### О чём работа
> Работа посвящена исследованию применения **методов машинного обучения** в прогнозировании рисков в страховании жизни
>> Осуществлена оценка потенциала машинного обучения для прогнозирования рисков в страховании жизни

### Реализация
В работе рассматриваются две модели:
- Модель Кокса (статистическая модель пропорциональных рисков)
- Random Survival Forest (модель машинного обучения)

Статистический анализ проводился с учетом:
- Цензурированных данных
- Временных рядов
- Множественных факторов риска

Машинное обучение реализовано через:
- Алгоритмы ансамблирования
- Методы оценки важности признаков
- Кросс-валидацию моделей

### Ограничения реализации
> **Формализация данных** и потеря содержательной информации
> 
> Необходимость **широкого набора компетенций** для работы с моделями
> 
> Сложности с **интерпретацией некоторых факторов**
> 
> Потребность в **качественных данных** для обучения моделей
